# Sculpin Bundle: Gist Embed

Allows for embedding GitHub Gists within a Sculpin website.

## Usage

Adds an extra Twig function that accepts the Gist ID and the filename as arguments.

```twig
{{ gist('the-gist-id', 'filename.txt') }}
```
